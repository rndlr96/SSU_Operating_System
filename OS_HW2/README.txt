프로세스 실행결과를 txt에 저장하고 싶다면 &>가 아닌 >를 사용
오류 메시지는 화면에 출력되고 실행결과만 저장된다.
&>로 하게 되면 오류 메세지만 txt에 저장되고 프로세스 실행 과정만이 출력된다.

실행 방법

1. 프로토콜에서 Makefile, c파일이 있는 디렉토리에 cd를 이용하여 이동한다.

2. make test를 입력하여 makefile을 적용하여 c파일을 실행한다.

3. 기본 설정으로 result1.txt에 결과가 저장되도록 하였으므로 result1.txt에 들어가 결과를 확인한다.

4. 4번째 인자인 우선순위를 변경하여 코드가 정상적으로 작동하는지 확인한다.
